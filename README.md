# Dnd-Tools

![Mr.Quest's Rollpad](https://img.shields.io/badge/Live%20Demo-Click%20Here-brightgreen?style=for-the-badge&logo=github&logoColor=white)

**Small but powerful browser-based tools for Dungeons & Dragons.**

**Live Tool:** [Mr.Quest's Rollpad](https://mrquestx.github.io/Dnd-Tools/)

---

## ✨ Features

### 🐉 Creature Rollpad
- Add and manage creatures for your encounters
- Create custom moves/attacks with dice expressions (e.g. `2d6+4`, `1d20+7`, `3d8+2d6+5`)
- Click-to-roll any move with clear results
- Drag & drop to reorder moves
- Save / Load encounters in browser
- Export / Import as JSON (great for sharing or backup)

### 📜 Custom Tables
- Create your own random tables (d6, d8, d20, etc.)
- Quick "Roll Table" button
- Expandable list of entries

### 🎲 Built-in Dice Roller
Supports standard D&D notation with modifiers and multiple dice.

---

## 🖥️ How to Use

1. Visit the **[live tool](https://mrquestx.github.io/Dnd-Tools/)**
2. Start adding creatures and their moves
3. Create custom tables for random events, loot, rumors, etc.
4. Roll directly in the app — perfect for quick DMing on a laptop or tablet

No installation needed. Works offline after first load.
